# Online-Blogging-Platform

A dynamic and user-friendly web application that allows users to create, edit, delete, and view blogs.
The platform includes a rich text editor, like and comment features, and complete profile management for each user.

Features
For Users

Create and publish blogs with a rich text editor (supports formatting, images, and links).

Edit or delete personal blogs anytime.

Read blogs from other users with an engaging layout.

Like and comment on blogs for interactive discussions.

Manage profile details, including avatar, bio, and user info.

Personalized dashboard to view all your posts and activities.

How It Works

User registers or logs in to the platform.

Create a new blog using the rich text editor.

Blogs are displayed in a feed with titles, images, and previews.

Users can like, comment, and interact with posts.
